# ENVÍOS Y LOGÍSTICA

**Palabras clave:** envío, entrega, códigos postales, estafeta, same day, express, CDMX

---

## TIPOS DE ENVÍO DISPONIBLES

### ENVÍO FULL / SAME DAY (CDMX)

| Característica | Detalle |
|----------------|---------|
| Entrega | El mismo día |
| Tiempo máximo | 180 minutos vía Uber |
| Origen | Desde CDMX |
| Disponibilidad | Lunes a domingo |
| Horario | 9:00 am a 1:00 pm |
| Restricción | **NO disponible en días festivos** |

#### Requisitos Específicos
- Solo direcciones dentro de códigos postales participantes
- Consultar archivo de códigos postales válidos
- Pedido confirmado antes de 1:00 pm
- Día hábil (no festivos)

#### Verificación Obligatoria ANTES de ofrecer Same Day
1. Confirmar código postal del cliente
2. Verificar en lista de códigos válidos
3. Confirmar horario actual (antes 1:00 pm)
4. Verificar que no sea día festivo

---

### ENVÍO EXPRESS - RESTO DEL PAÍS

| Característica | Detalle |
|----------------|---------|
| Cobertura | Todo México (excepto CDMX con Same Day) |
| Paquetería | Estafeta u otras equivalentes |
| Entrega | Siguiente día hábil |
| Tiempo estimado | 2 a 4 días hábiles |

**Slogan:** "¡Recíbelo rapidísimo en cualquier parte de México!"

#### Proceso
1. Pedido confirmado → Envío siguiente día hábil
2. Tiempo en tránsito: 1-3 días hábiles adicionales
3. Total: 2-4 días hábiles desde confirmación

---

## CÓDIGOS POSTALES VÁLIDOS PARA SAME DAY

### Verificación Requerida
- Consultar **SIEMPRE** antes de ofrecer Same Day
- Si el código postal **NO** está en la lista → Solo Express disponible

### Procedimiento de Verificación
1. Cliente solicita envío o menciona ubicación
2. Preguntar código postal específico
3. Consultar en archivo de códigos válidos
4. Confirmar disponibilidad de Same Day o ofrecer Express

---

## HORARIOS Y DISPONIBILIDAD

### Same Day CDMX
- **Lunes a Domingo:** 9:00 am - 1:00 pm
- **Días festivos:** NO disponible
- **Tiempo de entrega:** Máximo 180 minutos
- **Cut-off:** 1:00 pm (pedidos después → siguiente día)

### Express Nacional
- **Disponible:** Todos los días del año
- **Procesamiento:** Días hábiles únicamente
- **Envío:** Siguiente día hábil después de confirmación
